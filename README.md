# Query_KUSDT

This project is used to find the top 100 KUSDT holders and is divided into 3 levels as follows.

- Tier1 holds >= 10,000 KUSDT
- Tier2 holds < 10,000 && >= 5,000 KUSDT
- Tier3 holds < 5,000 && >= 1,000 KUSDT

# How to setup

- step 1: clone project
- step 2: run `npm i` in terminal
- step 3: run `node index.js` in terminal
